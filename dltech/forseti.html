ForseTi - Gerador de Fichas para Controle de Atendimento

O ForseTi é um sistema que gera fichas para controle de atendimento físico de usuários.
Atualmente está na versão pré alfa.
O sistema possui uma tela de login, apenas usuários cadastrados e ativos conectado no Active Directory da empresa.
Na área segura algumas opções de menu irão mudar dependendo do tipo do usuário:
- Usuário Vendas: Possui acessos limitados, tendo acesso apenas para "pegar" uma nova ficha retirada;
- Usuário ADM: Pode gerar relatórios e pode fazer tudo que o Usuário Padrão Faz.
- Usuário TI: Pode fazer tudo que os outros usuário faz e pode alterar o nome do Ponto de Atendimento (PA) caso seja preciso.

Sistema feito utilizando HTML5, Bootstrap, MySQL, AJAX e PHP

O sistema já imprime automaticamente a ficha para o cliente e quando ele é chamado pela TV é informado por um efeito sonoro.
O Sistema pode ser acessado via Github.
