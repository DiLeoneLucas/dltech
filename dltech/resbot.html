ResBot

O ResBot é um robô em Python que raliza resets em lote de Usuários do intune. Para isso abrimos o chrome com o Abre Chrome.bat para instanciar na porta correta do Selenium.
- @echo off start "" "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
O intune não interpreta o robô como um usuário, portanto, após algum tempo ele ira forçar a saída do usuário, para isso o sistema trava e assim ele solicita o login novamente, após isso ele irá voltar a resetar as senhas. O ResBot reseta a senha de um usuário em 15 segundos, dessa forma foi gerada uma automatização de 60% no reset de senhas dos usuários.
O robô pode ser acesado pelo Github.
